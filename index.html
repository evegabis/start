Flashcards de HTML

Projeto simples para estudar HTML e CSS

<html>

Tag principal da página. Tudo fica dentro dela.

<head>

Contém as configurações da página, como título e codificação.

<body>

Mostra todo o conteúdo visível do site.

<header>

Representa o cabeçalho da página.

<main>

Área principal onde ficam os conteúdos importantes.

<footer>

Rodapé da página com informações finais.
